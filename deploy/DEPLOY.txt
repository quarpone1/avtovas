Выгрузка статического сайта АвтоВас на сервер
============================================

Что заливать на хостинг
------------------------
В корень сайта (document root) положите файлы и папки:

  index.html
  styles.css
  script.js
  папка assets/ целиком (изображения, логотипы)

Структура путей относительная (./assets/...), менять домен в HTML не нужно.


Зависимости из интернета
------------------------
Страница подключает шрифты Google Fonts и GSAP с jsDelivr.
Сервер должен отдавать страницу по HTTPS; при блокировке CDN форма и анимации
могут работать частично — тогда перенесите скрипты локально в assets/.


Nginx
-------
См. файл deploy/nginx-site.example.conf — скопируйте и подставьте server_name и root.


Apache (общий хостинг)
----------------------
В корне проекта есть файл .htaccess — заливайте его вместе с index.html
(если хостинг Apache и разрешены переопределения).


Проверка после заливки
----------------------
  - Открывается главная, без 404 по CSS/JS/картинкам
  - Консоль браузера без ошибок загрузки
  - Якорные ссылки (#request, #trust) работают
